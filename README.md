# HShop
基于C#语言开发的桌面程序，采用HTML，CSS，JavaScript（VUE）做UI呈现
一个小店订单管理软件，已经集成了第三方电子面单打印，配送单打印，支持批量打印。支持快捷键操作。
软件采用MVC模式开发，当前版本支持本地数据库（sqlite 3.0）
